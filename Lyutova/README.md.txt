# Java Testing Tasks

Проект содержит решения трех задач на Java 11 для тестировщика.

## Задачи

1. **Операции с числами** - сравнение и арифметические операции
2. **Сравнение строк** - проверка идентичности строк
3. **Четные числа в массиве** - фильтрация четных чисел из массива

## Запуск

```bash
cd src
javac TestingTasks.java
java TestingTasks
